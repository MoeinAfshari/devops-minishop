# Basic Network Concepts

## Network

### Definition

A network consists of two or more computers that are linked in order to share resources (such as printers, CDs, ...), exchange files, or allow electronic communications.

### Real-World Example

The connection between my laptop, Wi-Fi modem, TV and my cell phone creates a network at home.

---

## LAN 

### Definition

A local area network (LAN) is a collection of devices connected together in one physical location, such as a building, office, or home.

### Real-World Example

The connection between classrooms and the office creates a LAN in the school.

---

## WAN 

### Definition

A wide area network (WAN) is a large computer network that connects groups of computers over large distances.

### Real-World Example

The Internet is the largest WAN.

---

## Internet

### Definition

The Internet is a global network of interconnected computer networks that communicate using standardized protocols.

### Real-World Example

Every connection to a website is a real-world example of the Internet like connection to github.

---

## Host

### Definition

A network host is a computer or other device connected to a computer network.

### Real-World Example

My laptop is a host right now.

---

## Node

### Definition

A node in networking refers to any device or point that connects to a network.

### Real-World Example

My laptop, WiFi modem and my cell phone act as nodes when connected together.

--- 

## Client

### Definition

The client refers to the device or software that initiates requests and interacts with the user.

### Real-World Example

Web browsers, email clients and mobile applications act as clients.

---

## Server

### Definition

A server is a dedicated computer that provides information, data, and services to other computers in a network, known as clients.

### Real-World Example

Examples of client-server models include many online gaming sites, email servers, and internet browsers.

---

## Packet

### Definition

In networking, a packet is a small segment of a larger message. Data sent over computer networks, such as the Internet, is divided into packets.

### Real-World Example

The `ping` command sends & receives packets.

---

## Frame

### Definition

A frame is the unit of transmission in a link layer protocol, and consists of a link layer header followed by a packet.

### Real-World Example

Every packet gets a frame in the router and then guides to the Internet network.
