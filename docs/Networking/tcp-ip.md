# TCP/IP Model

Layers of TCP/IP Model:
- Application Layer
- Transport Layer
- Internet Layer
- Network Access (Link Layer)

---

## 1. Application Layer

### Definition

This is the top layer of TCP/IP model, where applications interact with the network.

### Protocols

- HTTP (Hipertext Transfer Protocol)
- FTP (File Transfer Protocol)
- SMTP (Simple Mail Transfer Protocol)
- DNS (Domain Name System)

### Examples

- Web browsers
- email clients
- file-sharing tools

---

## 2. Transport Layer

### Definition

Ensure reliable and efficient delivery of data between devices, managing segmentation, ordering, and retransmission as needed.

### Protocols

- TCP (Transmission Control Protocol)
- UDP (User Datagram Protocol)

### Examples

- Downloading files, sending emails & ... use TCP
- Live video streaming, online gaming & ... use UDP

---

## 3. Internet Layer

### Definition

Responsible for addressing, packaging, and routing data packets so they can travel across networks and reach the correct destination device. It ensures that data can move between different networks efficiently.

### Protocols

- IP (Internet Protocol)
- ICMP (Internet Control Message Protocol)

### Examples

- Send packets from Source IP (192.168.1.10) to Destination IP (172.16.0.5)

---

## 4. Network Access (Link Layer)

### Definition

Responsible for physically transmitting data over network hardware, including cables, and wireless connections. It handles how data is formatted for the network medium and ensures it reaches the next device on the path.

### Protocols

- ARP (Address Resolution Protocol)
- SLIP (Serial Line IP)

### Examples

- Ethernet cables
- Fiber Optics
- Wi-Fi
