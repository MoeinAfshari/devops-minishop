# DevOps MiniShop

A simple Linux-based project for learning DevOps concepts step by step.

---

## Description

MiniShop is a practice project created during 120-day DevOps roadmap.

This project covers:

- Linux Administration
- Bash Scripting
- Git & GitHub Workflow
- Networking
- Docker
- Docker Compose
- Nginx
- Ansible
- CI/CD
- Monitoring
- Kubernetes

---

## Project Goals

Learn DevOps from beginner to production-ready level.

## How to Run

Clone repository

```bash
git clone <repository-url>
```

Go to project

```bash
cd devops-minishop
```

Run script

```bash
chmod +x scripts/server-info.sh
./scripts/server-info.sh
```

---

## Git Workflow

Each feature is developed in a separate branch.

Example:

```
feature/linux-documentation
feature/bash-refactor
feature/user-management
```

Every feature is merged into the `main` branch through a Pull Request.
